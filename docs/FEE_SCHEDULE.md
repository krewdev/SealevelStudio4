# Fee Schedule - Sealevel Studio

## Overview

Sealevel Studio uses SEAL tokens as the native utility token. All premium services require SEAL token payment.

## Free Trial (7 Days)

During the free trial period, users get access to:
- ✅ Arbitrage Scanner (50 scans)
- ✅ Transaction Simulator (30 simulations)
- ✅ AI Agent Queries (100 queries)
- ✅ Code Export (20 exports)
- ✅ Advanced Transaction Builder (50 transactions)

**Premium services are NOT included in the free trial.**

## Standard Pricing (SEAL Tokens)

### Existing Features (Freemium)

| Feature | Cost | Description |
|---------|------|-------------|
| Scanner Scan | 10 SEAL | Per scan operation |
| Scanner Auto-Refresh | 5 SEAL | Per hour of auto-refresh |
| Simulation | 20 SEAL | Per transaction simulation |
| AI Query | 5 SEAL | Per AI agent query |
| Code Export | 50 SEAL | Per code export |
| Advanced Transaction | 100 SEAL | Per advanced transaction build |

### Premium Services (NOT in Freemium)

#### Transaction Bundler

| Service | Cost | Description |
|--------|------|-------------|
| Multi-Send Transaction | 500 SEAL | Per multi-send transaction (up to 50 recipients) |
| Additional Recipient | 10 SEAL | Per recipient beyond base transaction |

**Example:**
- Send to 10 wallets: 500 SEAL (base) + 0 SEAL (within base) = **500 SEAL**
- Send to 60 wallets: 500 SEAL (base) + 100 SEAL (10 extra) = **600 SEAL**

#### Market Maker Agent

| Service | Cost | Description |
|--------|------|-------------|
| Setup | 2,000 SEAL | One-time setup fee for market maker agent |
| Monthly Subscription | 5,000 SEAL | Monthly subscription for market maker service |
| Per Trade | 50 SEAL | Per trade executed by the agent |

**Note:** Market maker agent operates with its own on-chain wallet and makes trading decisions based on analytics.

## Payment Methods

1. **SEAL Token Balance**: Deducted from user's SEAL balance
2. **Automatic Payment**: If balance insufficient, prompt for top-up
3. **Beta Tester Credits**: Applied automatically for eligible users

## Usage Tracking

All service usage is tracked and can be viewed in the account dashboard:
- Total SEAL spent
- Service usage statistics
- Remaining free trial credits
- Beta tester discount applied

## Notes

- Premium services require payment even during free trial
- Beta tester discounts apply automatically
- SEAL tokens can be purchased or earned through referrals
- All prices are in SEAL tokens (not SOL)
