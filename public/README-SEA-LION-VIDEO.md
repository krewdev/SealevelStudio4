# Sea Lion Animation Video

## File Location
Place your sea lion video file here:
- **File**: `sea-lion-animation.mp4` (or `.webm`, `.mov`, etc.)
- **Location**: `/public/sea-lion-animation.mp4`

## Recommended Specs
- **Format**: MP4 (H.264) or WebM (for better compression)
- **Size**: Under 5MB recommended for web
- **Dimensions**: 400-800px width (maintain aspect ratio)
- **Duration**: 3-10 seconds for looping
- **Background**: Transparent or dark background works best

## Supported Formats
- `.mp4` (H.264 codec) - Best compatibility
- `.webm` (VP9 codec) - Better compression
- `.mov` - Also supported but larger file size

## Usage
Once you place the video file, the SealAnimation component will automatically use it instead of the canvas animation.

The video will:
- Loop continuously
- Auto-play (muted)
- Display at the sizes configured in the component
